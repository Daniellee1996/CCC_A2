Subjectivity and Polarity of Twitter About Covid-19

Twitter about Covid-19 As mentioned above, we extracted two features from the full text of a tweet, polarity and subjectivity. Polarity is a float measures the emotional level given a text, ranging from -1 to 1 with -1 being most negative and 1 being most positive. Subjective, ranging from 0 to 1, measures how subjective it is when one is making a statement, where 0 represents objective factual information and 1 represents a private opinion. The main package we used to calculate the features is TextBlob, which only supports English narratives. Therefore, our backend would translate the text if a non-english language code is found.
